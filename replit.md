## Overview
A banking training platform in Portuguese that allows users to complete multiple training modules and obtain certificates. This is a static single-page HTML application.

## Project Architecture
- **Type**: Static HTML website
- **Structure**: Single `index.html` file containing all HTML, CSS, and JavaScript
- **Language**: Portuguese (Brazil)

## Development
- **Server**: Python's built-in HTTP server on port 5000
- **Command**: `python -m http.server 5000 --bind 0.0.0.0`

## Deployment
- **Type**: Static deployment
- **Public Directory**: Root (`.`)

## Features
- 8 training modules covering banking operations
- Progress tracking
- Certificate generation upon completion
- Responsive design
